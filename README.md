# steganography

Написать программу, позволяющую вставить секретное сообщение из файла *.txt в файл Microsoft Word, символы секретного сообщения в файле Microsoft Word выделяются цветом
Плюс простейшая обработка ошибок, если секретное сообщение невозможно поместить в MS Word, так как в том не хватает символов.

```sh
python program.py text.docx secret.txt
```
